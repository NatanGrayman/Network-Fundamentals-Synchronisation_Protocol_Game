This code consists of server, client and math class. 
1) The server and client need to be connected to the same host and port
2) Use a username and password pair, as shown in the appendix of the report. 
3) Once logged in, the game process will be managed by the server, where the math class is used to generate random questions and

4) The command and response pairs will be initiated between the two as indicated in the order shown in the appendix 
of the report.
